Overview

This is a simple and naive GUI for class STATIIP to present the results of homework projects.

All the website is built under Flask frame using Python. Several javascript libraries are adopted for better visualization effects.

All the homework projects are written in Python, and there might exist tons of bugs in these codes. Therefore please feel free to click the contact me button in footnote and report the bugs.
